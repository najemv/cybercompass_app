# CybercompassApp

Simple REST API server to handle requests from clients.

## Setup

Install Python 3.6 and newer

`pip install Django`

`pip install djangorestframework`

You will also need Pillow lib for image support. Use this:

`python -m pip install Pillow`

For more informations, check this: https://www.django-rest-framework.org/

Go to repo directory and run:
`python .\manage.py runserver`


## Features
Adding new data to database through admin panel.

Retrieving data through API requests.

Session and basic authentication.
